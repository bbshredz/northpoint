Connection failed — CORS or network error
Your browser can't reach FreeScout directly due to CORS restrictions or because you're not on the internal network. Make sure you're connected to VPN, then add the following to your FreeScout web server config:
# Nginx — add inside the server {} block add_header 'Access-Control-Allow-Origin' 'https://northpoint.nacs.space'; add_header 'Access-Control-Allow-Headers' 'Authorization, Content-Type'; add_header 'Access-Control-Allow-Methods' 'GET, OPTIONS'; add_header 'Access-Control-Allow-Credentials' 'true';
# Apache — add to .htaccess or VirtualHost Header always set Access-Control-Allow-Origin "https://northpoint.nacs.space" Header always set Access-Control-Allow-Headers "Authorization, Content-Type"
After adding CORS headers, restart nginx/Apache and try connecting again.
